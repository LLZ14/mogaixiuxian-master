## 转移存档            
![转移存档](pic/photo4.jpg)     
存档位置  
```
\Yunzai-Bot\plugins\xiuxian-emulator-plugin\resources\data
```      
若要转移存档，将上面画框的文件保存，将修仙插件删除，执行上面安装命令，将上面画框的文件替换到对应文件,然后执行【#一键同步】
